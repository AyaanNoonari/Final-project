Html CheatSheet

1. Variables and Data Types

let and const for variable declaration.

Primitive data types: string, number, boolean.

2. Functions

Define a function with function functionName() {}.

Arrow functions: const add = (a, b) => a + b;

3. Arrays

Create an array: const myArray = [1, 2, 3];

Array methods: push(), pop(), forEach().

4. Objects

Create an object: const myObject = { key: 'value' };

Access object properties: myObject.key;

5. Conditionals

if, else if, else statements.

Conditional (ternary) operator: condition ? trueExpr : falseExpr;

6. Loops

for, while, and do-while loops.

7. DOM Manipulation

Select elements: document.getElementById('id').

Manipulate HTML content: element.innerHTML = 'new content';

8. Events

Event listeners: element.addEventListener('event', callback);

9. AJAX and Fetch

Fetch API for making HTTP requests.

Example: fetch('https://api.example.com/data')

10. Error Handling

try, catch, finally blocks for error handling.

11. Local Storage

Store data locally: localStorage.setItem('key', 'value');

12. ES6+ Features

Destructuring, spread/rest operators, template literals.

13. Promises

Create a promise: new Promise((resolve, reject) => { /* ... */ });

14. Async/Await

Asynchronous programming with async and await.

15. Modules

Organize code into modules using import and export.
